# NLP Assignment 2

This project compares three models for sentiment classification on the IMDB dataset:

- Baseline (TF-IDF + Logistic Regression)
- LSTM
- CNN

## Results
- Baseline: 0.8950
- LSTM: 0.8885
- CNN: 0.8909

## Conclusion
The baseline model achieved the best performance, showing that traditional NLP techniques remain highly effective for this task.
