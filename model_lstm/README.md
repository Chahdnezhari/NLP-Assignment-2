# LSTM Model

## Model Architecture
This model uses a Long Short-Term Memory (LSTM) network:
- Embedding layer
- LSTM layer (captures sequential dependencies)
- Dense layer with sigmoid activation

## Techniques Applied
- Text preprocessing (cleaning, tokenization, padding)
- Sequential modeling with LSTM
- Word embeddings
- Binary classification using sigmoid activation
- Adam optimizer and binary cross-entropy loss

## Performance
Accuracy: 0.8885
